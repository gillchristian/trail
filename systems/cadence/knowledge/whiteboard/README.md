# whiteboard/

Discussions in flight — explored but not committed to action. Not a backlog: entries
record *what we considered* so future sessions don't reopen settled ground. One file per
topic: `<topic-slug>.md` with a Status line (open / resolved / tabled), Context, Options
considered, Where we landed, Follow-ups. When a topic resolves: promote to an ADR, a
backlog task, or a brief update — and update the Status line; keep the entry.
